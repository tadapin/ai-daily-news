How coding agents work

コーディングエージェントは、大規模言語モデル（LLM）を基盤にしており、見えないプロンプトや呼び出し可能なツールを使ってLLMの機能を拡張するソフトウェアです。LLMはテキストをトークンという整数列に変換して処理し、トークン数に応じて課金や処理制限があるため、この仕組みの理解が重要です。近年のモデルはマルチモーダル対応で、画像などの非テキスト入力もトークン化して同様に処理可能です。これにより、コーディングエージェントはより複雑で多様なタスクを実行できます。

What is agentic engineering?

エージェント工学（agentic engineering）とは、コードを書き実行できるコーディングエージェントの支援を受けてソフトウェアを開発する手法を指します。エージェントは大規模言語モデル（LLM）を用いて目標を設定し、コードを生成・実行しながら目的達成に向けて反復処理を行います。コードの実行能力があることで、生成されたコードの動作検証が可能となり、実用的なソフトウェア開発が実現します。人間の役割は、最適な解決策を選び出し、エージェントに適切な指示やツールを提供することにあります。
